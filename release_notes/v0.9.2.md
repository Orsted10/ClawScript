# VoltScript v0.9.2 Release Notes
Release Date: 2026-02-15

## Summary
VoltScript v0.9.2 strengthens the performance pipeline with inline caches and an optional LLVM AoT build path, adds new performance math helpers, and expands documentation for the updated build and runtime workflows.

## New Features
- Inline caches for global lookups, property access, and call dispatch in the VM.
- Optional LLVM AoT pipeline with object emission and linker integration.
- New performance math helpers: fibFast(n) and arraySumFast(n).
- CLI support for --aot-output to emit native object files.

## Improvements
- Expanded documentation for AoT builds and performance helpers.
- Updated versioning across README and docs for v0.9.2.

## Bug Fixes
- Improved VM call handling around cached call sites and closures.

## Breaking Changes
- None.

## Migration Steps
- For AoT builds, install LLVM 16+ and configure with -DVOLT_ENABLE_AOT=ON.
- Use --aot-output=main.o script.volt to emit an object file and link with the runtime library.
